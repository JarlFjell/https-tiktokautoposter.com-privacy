Datenschutzerklärung

Diese Datenschutzerklärung beschreibt, wie die App TikTok AutoPoster personenbezogene Daten verarbeitet.

1. Verantwortlicher

Verantwortlich für die Datenverarbeitung ist der Betreiber der App TikTok AutoPoster.

2. Erhobene Daten

Die App verarbeitet ausschließlich Daten, die für die Nutzung der TikTok API erforderlich sind, darunter:

  • OAuth‑Token, die von TikTok bereitgestellt werden
  • Vom Nutzer ausgewählte Videos und Metadaten

Es werden keine zusätzlichen personenbezogenen Daten gesammelt.

3. Zweck der Verarbeitung

Die Daten werden ausschließlich verwendet, um Inhalte im Namen des Nutzers über die TikTok API zu veröffentlichen.

4. Speicherung

Die App speichert keine personenbezogenen Daten dauerhaft. OAuth‑Token werden nur temporär auf dem Gerät des Nutzers gehalten.

5. Weitergabe an Dritte

Es erfolgt keine Weitergabe von Daten an Dritte.

6. Rechte der Nutzer

Nutzer können ihre TikTok‑Berechtigungen jederzeit über die TikTok‑Einstellungen widerrufen.

7. Kontakt

Bei Fragen zur Datenverarbeitung kann der Nutzer den App‑Entwickler kontaktieren.
